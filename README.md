# IEEE 754 Simulator

**Author:** Emily Lister  
**Course:** Computer Systems
**Language:** Java  

---

## Project Description

This program simulates the **IEEE 754 32-bit floating-point standard**, including **binary encoding, addition, and multiplication** of decimal numbers.  

The program:

- Converts decimal numbers to **IEEE 754 32-bit binary format**  
- Performs **addition** or **multiplication** on two decimal operands  
- Displays the binary representations of operands and the result  

This project demonstrates concepts in:

- Floating-point representation in computers  
- Binary arithmetic and normalization  
- Manual IEEE 754 encoding  

---

## How to Compile

Use the Java compiler in your terminal:
javac IEEE754Simulator.java

## How to Run
java IEEE754Simulator

The program will prompt for:
- Operand 1: Decimal number (e.g., 2345.125)
- Operand 2: Decimal number (e.g., 0.75)
- Operator: add or mult

It will then output the IEEE 754 binary representations of both operands and the result.

## Features
- Converts decimal floats to IEEE 754 32-bit binary
- Supports addition and multiplication operations
- Illustrates binary mantissa, exponent, and sign bits
- Fully implemented in Java for educational purposes
